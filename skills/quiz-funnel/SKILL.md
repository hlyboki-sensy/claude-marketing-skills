# Quiz Funnel v2.0 — генератор квіз-воронок

Скіл аналізує продукт, його аудиторію та створює повне технічне завдання на персоналізований квіз-лендинг. Формула побудована на аналізі 15+ висококонверсійних квіз-воронок (NeuroSamurai, Noom, Focus & Flow, Done., BedGear, RANAVAT та ін.), дослідженнях ADHD-UX і гейміфікації.

## Що робить

1. Збирає інформацію про продукт через діалог
2. Рекомендує оптимальний формат квіза з 4 варіантів
3. Аналізує аудиторію (болі, тригери, архетипи, трансформації)
4. Генерує повну структуру квіза з ADHD-friendly UX
5. Видає готове ТЗ у markdown — можна одразу дати Claude Code на реалізацію

## 4 формати квізів

| Формат | Екранів | Час | Найкраще для |
|--------|---------|-------|-----------------|
| **A. Діагностика** | 20+ | 3-5 хв | Курси, спринти, челенджі |
| **B. Типологія** | 12-15 | 2-3 хв | Блоги, інфо-продукти, digital |
| **C. Ультра-скринінг** | 5-7 | 1 хв | Консалтинг, SaaS, послуги |
| **D. Калькулятор** | 8-12 | 1-2 хв | Детокс, продуктивність, фінанси |

## Використання

```
/quiz-funnel Створи квіз для мого 21-денного курсу з продуктивності
/quiz-funnel Типологічний квіз для ADHD-блогу
/quiz-funnel Калькулятор вартості прокрастинації для застосунку
/quiz-funnel Швидкий скринінг для запису на консультацію
```

## Як працює

Скіл проходить 5 фаз (деталі в PROMPT.md):
- **Фаза 1**: Збір інформації про продукт (інтерактивно)
- **Фаза 1.5**: Рекомендація та вибір формату квіза
- **Фаза 2**: Аналіз ДНК аудиторії (болі, тригери, архетипи)
- **Фаза 3**: Генерація структури квіза за формулою 10 патернів
- **Фаза 4**: Проєктування скорингу та калькулятора
- **Фаза 5**: Збірка повного ТЗ у markdown-файл

## База знань

- `PROMPT.md` — повна інструкція з генерації квіза
- `references/formula.md` — формула 10 патернів і карти екранів для всіх форматів
- `references/formats.md` — детальні специфікації 4 форматів
- `references/knowledge-base.md` — конверсійні патерни, ADHD-UX, дизайн-системи, CTA-бібліотека, кейси з метриками
