# Плейбуки загонів — готові набори під типові задачі

Це **стартові шаблони**, не догма. Підлаштовуй під конкретний ЦКП, прибирай зайве, додавай потрібне.
Формат: **Тригер → Загін (ролі) → Потік → Вихід.** `Ω` = власний стек/скіл користувача (пріоритет).

> **Примітка.** Частина `Ω`-скілів у прикладах нижче (напр. `client-dna-v2`, `viral-reels`, `instagram-stories`, `page-cro`, `senior-marketing-analyst`) — це ілюстрації, і вони **не входять** у цей набір. Заміни їх на свої аналоги або встанови окремо. Скіли, що є в наборі: `content-marketing`, `carousel-from-video`, `viral-threads`, `tone-of-voice`, `competitive-matrix`, `deep-competitor-research`, `analyst-deck`, `meta-ads-pro`, `quiz-funnel`, `nbp-skill`, `research-zettelkasten`, `council`, `humanizer-ua`, `skill-conductor`.

---

## 1. 🚀 Запуск продукту / оффера (маркетинг)
**Тригер:** «запусти оффер», «зробити запуск», «вивести продукт».
**Загін:**
- Лід: Ω `senior-marketing-analyst`
- Аудиторія: Ω скіл `client-dna-v2` → **Offer & Lead Gen Strategist**
- Позиціювання/бренд: **Brand Guardian**, Ω `tone-of-voice`
- Канали: **Email Marketing Strategist**, **Paid Social Strategist**, Ω `meta-ads-pro`, **Content Creator**
- Лендінг: Ω `page-cro` → **Frontend Developer** + **Tracking & Measurement Specialist**
- Критик: **Reality Checker** + (опц.) `council`
**Потік:** client-dna → [оффер ∥ позиціювання] → [канали ∥ лендінг] → критик → синтез.
**Вихід:** оффер + лендінг + канальні матеріали + план запуску.

## 2. 🔬 CRO-аудит лендінга / сторінки
**Тригер:** «чому не конвертить», «аудит лендінга», «підняти конверсію».
**Загін:** **Persona Walkthrough Specialist** (психологія скролу) + Ω `page-cro` + **Frontend Developer** (тех.фікси) + **Tracking & Measurement Specialist** + **Accessibility Auditor** + критик **Evidence Collector** (скрін-пруфи).
**Потік:** паралельний аудит за лінзами → дедуп → пріоритезація фіксів → (опц.) впровадження.
**Вихід:** звіт з пріоритезованими фіксами + проєкт правок.

## 3. 🕵️ Глибоке дослідження конкурентів / ринку
**Тригер:** «розбери конкурентів», «аналіз ринку», «досьє на конкурента».
**Загін:** Ω `senior-marketing-analyst` (лід) → Ω `market-researcher` ∥ **Trend Researcher** ∥ **Business Strategist** → Ω скіл `deep-competitor-research` / `competitive-matrix` → Ω `data-analyst` + **Analytics Reporter** → Ω `slide-builder`.
**Потік:** fan-out збір фактів → синтез → матриця → дека. Для 50+ джерел → `research-zettelkasten` або `Workflow`.
**Вихід:** структуроване досьє + конкурентна матриця + презентація.

## 4. 🎬 Контент-машина (рілс / карусель / сторіс)
**Тригер:** «настругай контенту», «серія рілс/каруселей під клієнта».
**Загін:** Ω клієнтський `tone-of-voice` + `client-dna` → Ω скіли `viral-reels` / `carousel-from-video` / `instagram-stories` → **Content Creator** ∥ **Visual Storyteller** + **Video Optimization Specialist** → Ω `humanizer-ua` (фінал).
**Потік:** контекст клієнта → паралельна генерація форматів → гуманізація → типографіка.
**Вихід:** пакет готового контенту + збереження в Notion/past-research.

## 5. 🌐 Сайт / застосунок під ключ
**Тригер:** «зроби сайт», «збери застосунок», «MVP».
**Загін:** **Software Architect** / **Backend Architect** (лід) + **Frontend Developer** + **UI Designer** + **DevOps Automator** + критики **Code Reviewer** + **QA / Test Results Analyzer** + **Application Security Engineer**.
**Потік:** архітектура → [бек ∥ фронт ∥ UI] конвеєром → рев'ю+QA+security → деплой (vercel).
**Вихід:** робочий застосунок + рев'ю + деплой. (Складне → `Workflow` з worktree-ізоляцією.)

## 6. 💰 Фінмодель / юніт-економіка / прайсинг
**Тригер:** «порахуй юніт-економіку», «фінмодель», «ціноутворення».
**Загін:** **Financial Analyst** (лід) + **FP&A Analyst** + **Pricing Analyst** + Ω `data-analyst` + **Analytics Reporter** + критик **Model QA Specialist**.
**Потік:** збір вхідних → модель ∥ прайсинг → перевірка → візуалізація.
**Вихід:** модель + сценарії + рекомендація по ціні.

## 7. 🎨 Бренд з нуля
**Тригер:** «розроби бренд», «бренд-платформа».
**Загін:** `council` (стратегія) → **Brand Guardian** (лід) + **Business Strategist** + **UI Designer** + **Visual Storyteller** + **Content Creator** + Ω `tone-of-voice`.
**Потік:** Рада задає напрям → платформа бренду → візуал ∥ вербальна ідентичність → гайдлайни.
**Вихід:** бренд-платформа + ToV + візуальні принципи.

## 8. 📱 SMM під клієнта (Instagram/LinkedIn)
**Тригер:** «веди інсту клієнта», «контент-план на місяць».
**Загін:** Ω клієнтська база + `tone-of-voice` → **Instagram Curator** / **LinkedIn Content Creator** + Ω `content-marketing` + `instagram-stories` → **Content Creator** → Ω `humanizer-ua`.
**Потік:** контекст → контент-план → пости/сторіс → гуманізація → Notion/past-research.
**Вихід:** контент-план + готові одиниці контенту.

---

### Як комбінувати
Реальна задача часто = 2-3 плейбуки поспіль (напр. дослідження конкурентів → запуск оффера → контент-машина).
Веди їх **фазами**: завершив фазу → показав користувачеві проміжний результат → наступна фаза. Не зливай усе в один хаотичний запуск.
