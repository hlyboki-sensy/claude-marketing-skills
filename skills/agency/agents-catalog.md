# Реєстр агентів «Агенції»

> Цей каталог індексує зовнішній набір **agency-agents** (232 агенти, 16 дивізіонів) — він встановлюється окремо: https://github.com/msitarzewski/agency-agents . Разом із ним «Агенція» диригує маркетинговими скілами з цього набору.
> **Як кастувати:** для маркетингових задач спершу бери профільний скіл із цього набору (список нижче), далі добирай спеціалістів agency для ширини й паралельної роботи.
> `subagent_type` у виклику `Agent` = **ім'я агента жирним** (як у списку типів). Якщо точна назва не спрацювала — підбери найближчу за списком типів агентів.
