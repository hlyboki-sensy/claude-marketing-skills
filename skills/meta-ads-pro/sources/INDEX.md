# meta-ads-pro — провенанс источников

Скилл построен на **актуальной методологии Ben Heath** (Meta Ads, эра post-Andromeda).

Каждый факт в базе знаний помечен ссылкой на первоисточник в формате `⟦video_id @дата⟧`. Это публичные видео — открываются по адресу `https://www.youtube.com/watch?v=<video_id>`.

Такая разметка даёт:
- **проверяемость** — любое утверждение можно сверить с первоисточником;
- **актуальность** — по дате видно, насколько свежая тактика и не устарела ли она после обновлений платформы;
- **отсутствие галлюцинаций** — числа и факты берутся только из источников, а не «из головы».
