# Глосарій Meta Ads

> Швидкий довідник термінів. Деталі — у відповідних розділах (посилання нижче).

## Доставка / алгоритм
- **Andromeda** — retrieval-движок Meta (анонс груд.2024), персоналізація доставки на рівні людини. [[00-philosophy/andromeda-era]]
- **Retrieval / ranking** — етапи відбору оголошення з мільйонів кандидатів. [[00-philosophy/how-delivery-works]]
- **Learning phase / Learning limited** — фаза навчання адсета; поріг ~50 результатів/тиждень (не абсолют). [[05-optimization/learning-phase]]
- **Auction overlap** — власні дублі-кампанії конкурують в аукціоні → недодоставка. [[01-structure/campaign-structure]]
- **Ad-sequencing** — Meta сама веде людину top→mid→bottom funnel в одному адсеті. [[00-philosophy/andromeda-era]]

## Структура / бюджет
- **CBO / ABO** — бюджет на кампанію / на адсет. [[04-budget-scaling/big-spend-scaling]]
- **Advantage+ (ASC → Advantage Plus Sales)** — автоматизовані кампанії. [[01-structure/asc-vs-manual]]
- **Controls vs Suggestions** — hard-межі (локація) vs м'які підказки (інтереси). [[01-structure/campaign-structure]]
- **Scaling ceiling** — денний бюджет, де кампанія ще прибуткова. [[04-budget-scaling/big-spend-scaling]]

## Таргетинг
- **Broad / open targeting** — без детального таргету, Meta сама. [[03-targeting/broad-vs-detailed]]
- **Value Rules** — бід +/− за демографію під твою юніт-економіку. [[03-targeting/value-rules]]
- **Advantage+ Audience / Lookalike** — авто-аудиторія / схожі. [[03-targeting/advantage-plus-audience]]
- **Audience segments (new/engaged/existing)** — розбивка warm vs cold. [[09-retargeting/retargeting-post-andromeda]]

## Креатив / копірайт
- **Hook** — перші ~3 сек відео; найвищий важіль. [[06-creatives/hooks]]
- **Ad angle** — причина купити; один angle/оголошення. [[07-copywriting/ad-angles]] · [[06-creatives/why-ads-fail]]
- **Creative enhancements** — AI-апгрейди Advantage+ (будь фільтром). [[06-creatives/creative-enhancements]]
- **Creative Testing tool** — роздільний тест схожих креативів. [[12-testing/post-andromeda-testing]]
- **UGC / Influencer** — контент без/з аудиторією. [[06-creatives/ugc-ai]]

## Метрики / вимірювання
- **CPM / CPC / CTR / CPL / CPA / ROAS / Frequency** — формули й інтерпретація. [[guardrails/metric-definitions]]
- **Health Score** — 5-компонентна оцінка адсета. [[10-measurement/diagnostics]]
- **Significance / min data** — коли можна судити про результат. [[10-measurement/significance-thresholds]]
- **CAPI / attribution / engage-through** — матчинг подій, вікна. [[10-measurement/capi-event-match]]
- **Incremental attribution** — конверсії, що не сталися б без реклами. [[03-targeting/value-rules]]
