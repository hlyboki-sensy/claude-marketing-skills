# Маркетингові скіли для Claude Code

Набір скілів (skills) для [Claude Code](https://claude.com/claude-code), які перетворюють Claude на маркетингову команду: аналіз конкурентів і ринку, контент, реклама, каруселі, дослідження, стратегія та фінальна чистка українських текстів.

Скіли не потребують коду — це структуровані інструкції, які Claude сам підхоплює під потрібну задачу. Достатньо покласти теки у свій каталог скілів (нижче — як саме).

---

## Що всередині

| Скіл | Що робить |
|---|---|
| **content-marketing** | Контент-конвеєр: перетворює вихідний матеріал (текст, транскрипцію, вебінар) у готові формати для соцмереж. |
| **carousel-from-video** | Каруселі для Instagram і LinkedIn — обкладинка, слайди, структура, підпис, CTA. |
| **viral-threads** | Вірусні пости для Instagram Threads — хуки, формати, серії. |
| **tone-of-voice** | Зворотний інжиніринг Tone of Voice з будь-якого тексту: 8-рівневий аналіз і готовий ToV-документ. |
| **competitive-matrix** | Конкурентна матриця (функції, ціни, позиціонування) з візуальним виходом. |
| **deep-competitor-research** | Глибоке досьє на конкретного конкурента: продукт, маркетинг, сильні/слабкі сторони. |
| **analyst-deck** | Аналітична презентація (.pptx) senior-рівня з досліджень і даних. |
| **meta-ads-pro** | Експертний AI-таргетолог по Meta Ads (Facebook/Instagram): аудит кабінету, структура, оптимізація. |
| **quiz-funnel** | Персоналізовані квіз-воронки під будь-який продукт (діагностика, типологія, калькулятор). |
| **nbp-skill** | Промпт-інжиніринг для Nano Banana Pro (Gemini 3 Pro Image): візуали, інфографіка, слайди. |
| **research-zettelkasten** | Каркас для довготривалих досліджень за методологією Zettelkasten (джерела → нотатки → синтез). |
| **council** | «Рада» — 5 незалежних AI-лінз стрес-тестують важливе рішення й дають спільний вердикт. |
| **humanizer-ua** | Прибирає ознаки ШІ-генерації з українського тексту: канцелярит, кальки, шаблони. |
| **skill-conductor** | Каркас для проєктування й тестування власних скілів (eval-цикл). |
| **agency** | Оркестратор-«Агенція»: збирає під задачу команду AI-агентів і зводить результат докупи. |

## Встановлення

Скіли живуть у теці `~/.claude/skills/`. Є два способи їх туди покласти.

### Спосіб 1 — клонувати репозиторій

```bash
git clone https://github.com/hlyboki-sensy/claude-marketing-skills.git
cp -R claude-marketing-skills/skills/* ~/.claude/skills/
```

### Спосіб 2 — завантажити ZIP

1. Натисніть зелену кнопку **Code → Download ZIP** угорі сторінки.
2. Розпакуйте архів.
3. Скопіюйте всі теки з папки `skills/` у `~/.claude/skills/` (створіть її, якщо немає).

Після цього перезапустіть Claude Code — скіли з'являться автоматично.

## Як користуватися

Claude сам обирає потрібний скіл за суттю запиту — просто опишіть задачу («зроби конкурентну матрицю», «почисти цей текст українською», «збери квіз-воронку»). Також можна викликати скіл напряму через `/назва-скіла`.

Скіли розраховані на українськомовну роботу; частина довідкових матеріалів — російською (робочі конспекти методологій). Тексти на виході рекомендується проганяти через **humanizer-ua**.

---

## Що НЕ входить

Це відібраний набір авторських та узагальнених інструментів. Свідомо не включені скіли, авторство яких належить третім особам, а також будь-що з реальними клієнтськими чи персональними даними. Скіл **agency** індексує зовнішній набір [agency-agents](https://github.com/msitarzewski/agency-agents) — його за потреби встановлюють окремо.
